LiveCuts v1.2 系统流程图

飞书表导入 → 结构化线索 → 选品 → 可视化粗找 + 精剪 → 真值复用

PHASE 1 数据导入
📋 飞书表 (xlsx)
主播每场直播后更新的商品+素材表
列: 款号、品名、单价、素材(时间线索)、状态...
素材列格式极乱: 全角/半角混用、换行/分号分隔
📤 拖拽上传 xlsx
XlsxDropZone 组件，支持拖放或点击选择
ImportPage
⚙️ xlsx_parser.py 解析
移植自 v1.1 material-parser.ts
1. 全角→半角，标准化分隔符
2. 按 / 拆分为 fragments
3. 每个 fragment 提取: 日期(X月X日)、时间点(HH:MM)、场次(第二场)、主播(XX老板)
4. SHA1 去重 hash + 置信度(HIGH/MEDIUM/LOW)
🔄 diff_service.py 变更对比
对比新数据与数据库现有数据
4 类变更: 新增SKU / 新增线索 / 状态变更 / 链接变更
已存在的 lead (相同 hash) 自动跳过
💾 SQLite 存储
products 表 + leads 表 + xlsx_snapshots + import_diffs
products: SKU 商品信息 (UPSERT)
leads: 线索记录 (UNIQUE: sku_code + year + fragment_hash)
xlsx_snapshots: 文件快照 (按 MD5 去重)
import_diffs: 变更日志
📊 DiffPanel 展示变更
4 个 Tab 分类展示本次导入的变更明细
ImportPage
用户在此确认导入结果是否正确。可反复导入不同版本的表格，系统自动增量合并。
PHASE 2 选品计划
🛍️ 左栏: 可选 SKU 列表
显示所有 SKU，可搜索筛选
每个卡片: SKU图 + 款号 + 品名 + lead数
点击 "+" 加入右栏
PlanPage
📅 右栏: 今日计划
已选 SKU 有序列表
可拖拽排序
状态切换: 待处理 → 进行中 → 已完成
点击 × 移除
PlanPage
用户每天在此选定要切片的 SKU。计划按日期存储，可查看历史。
PHASE 3 核心审核 — 粗找 + 精剪
🎯 选择 SKU + Lead
左侧栏选中一个 SKU，再选一条 Lead 线索
Lead 显示: 日期 + 时间点 + 置信度
例: "1月4日 03:01, 03:15" (HIGH)
ReviewPage
用户还需要在顶部输入代理视频文件路径（后续可优化为自动匹配日期→视频文件）
📸 批量抽帧 (ffmpeg)
从代理视频抽取帧画面
使用 -ss 在 -i 前 (keyframe seek, ~50ms/帧)
并发 semaphore=4，帧缓存到 data/frames/
已存在的帧直接跳过
🔍 Level 1: ±15min 粗找
以线索时间点为中心，每 2min 抽 1 帧 ≈ 15 帧网格
帧网格展示，每帧带时间戳，hover 可放大
CoarseGrid
❓ 在 ±15min 内找到目标了吗？
没找到 → 扩大范围
🔭 Level 1-wide: ±45min
每 5min 抽 1 帧 ≈ 18 帧
覆盖更大时间范围
看到疑似帧 → 点击放大
🔬 Level 3: ±2min 精细
以点击帧为中心，每 10s 抽 1 帧 ≈ 24 帧
精确定位目标出现的时刻
在 Level 3 点击确认帧 → 进入精剪
✂️ 精剪 FineTrimmer
命中点前后 90s 的帧条带 + 视频预览播放器
帧条带: 1帧/秒 ≈ 180帧 水平滚动
绿色滑块=起点, 红色滑块=终点, 拖拽设置
快捷键: [ 设起点, ] 设终点, 空格播放, ←→ ±1s
时间显示: 02:15:30 - 02:18:45 (3m15s)
FineTrimmer
⭐ 评分 + 标签
1-5 星评分 + 预设标签
标签: 远景 / 细节 / 试穿 / 讲得顺 / 情绪好 / 商品全屏
💾 保存 verified_clips
真值片段入库
记录: sku_code, video_path, start_sec, end_sec,
rating, tags, lead 原始时间, 偏移量
自动生成封面帧 (thumbnail)
↩ 回到粗找，继续处理下一条 Lead 或下一个 SKU
PHASE 4 检索复用
🔎 按 SKU 检索
输入款号或品名搜索
结果: 商品卡片 + Lead 时间线 + 真值片段列表
真值按评分排序，可播放预览
SearchPage
📆 按日期检索
选择日期查看当天所有 SKU
结果: 当天 SKU 列表 + 每个 SKU 的时间点 + 真值统计
可快速加入今日计划
SearchPage
